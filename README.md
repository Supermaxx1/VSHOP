Vshop
Vshop is a modern, user-friendly stock management and retail sales application built in Flutter for Windows and web platforms.
Designed for small businesses, retail shops, and student developers, Vshop streamlines inventory management, sales tracking, and reporting for day-to-day operations with an intuitive interface.
